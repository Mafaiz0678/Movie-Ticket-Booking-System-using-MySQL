Designed and implemented a relational MySQL database for a Movie Ticket Booking System, managing data for movies, theaters, screens, show timings, seats, and bookings.
Structured and supervised normalized tables for movies, theaters, showtimes, and customer bookings, ensuring strong referential integrity through primary and foreign keys.
Developed and executed optimized MySQL queries using JOINs, constraints, and aggregations to retrieve movie schedules, theater-wise show listings, and booking details.
Automated booking workflows by tracking showtime availability, ticket counts, and booking timestamps, improving transaction accuracy and consistency.
Improved operational efficiency by streamlining ticket reservations and show management, reducing manual intervention by 65% and enhancing overall system scalability.
